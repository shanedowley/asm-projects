# ZX Spectrum Asteroids Clone

## Building

From `src/` run:

```
pasmo --tapbas asteroids.asm ../build/asteroids.tap
```

## Running in Emulator (zxsp)

```
cd build
zxsp asteroids.tap
```

While running:
- Controls display in-game; press `1` to start if needed.
- Use `O/P` rotate, `Q` thrust, `M` fire, `SPACE` warp.
